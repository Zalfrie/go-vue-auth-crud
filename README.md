# Go-Vue Auth CRUD Application

Full-stack example with Golang backend & Vue.js frontend.

## Prerequisites
- Go >=1.18
- Node.js >=16
- MySQL

## Getting Started

### Clone & Initialize
```bash
git clone https://github.com/yourusername/go-vue-auth-crud.git
cd go-vue-auth-crud

#Frontend Setup

cd ../frontend
cp .env.example .env
npm install
npm run serve
